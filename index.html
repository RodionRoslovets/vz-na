Для вычисления нужно ввести количество параметров в наборе и шаг вычисления — точность. Шаг вычисления должен быть числом которому кратна сумма ряда параметров — это единица.

Также можно сделать отсев вычисленного массива параметров с помощью задания системы неравенств — строки для фильтрации значений. Для этого нужно ввести строку, соответствующую следующим параметрам:

  • Неравенство задаётся символами: <, >, <=, >=
  • Неравенство обособлено пробелами от аргументов
  • Возможно ввести несколько аргументов, через запятую
  • Аргументы — номера элементов в наборе
  • Несколько правил записываются через запятую и пробел после неё

Пример: 1,4 > 2, 2 <= 3 — 1 и 4 параметр больше второго, второй меньше или равен третьему.

Также можно подсчитать общий вес набора параметров, для этого можно ввести последовательность весовых коэффициентов — насколько элемент важен в общей сумме набора.

Вводятся они одной строкой через запятую и пробел, их должно быть столько же сколько и параметров в наборе.

Пример для четырёх параметров: 1, 0.5, 0.7, 0.2

Как видно из примера десятичную дробь можно записать через запятую.
